## コマンドの説明

1. ls - ファイルやディレクトリの情報を表示する
1. cd - change direction
1. cp - copy file
1. mv - move file
1. mkdir - make directory
1. rm - remote file
1. pwd - print current directory
1. rmdir - remove directory
1. cat - print file content
1. more/less - paging file content
1. find - find file
1. which - print file path
1. man - print manual
